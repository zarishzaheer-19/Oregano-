🍕 Oregano Restaurant Website

A modern and responsive restaurant website for Oregano, designed to provide customers with a smooth food ordering experience. The website features a clean UI, special deals, a menu section, and an easy-to-use layout that works on both desktop and mobile devices.

✨ Features

- 🍕 Modern and responsive design
- 📋 Restaurant menu with food categories
- 🔥 Special deals and promotions section
- 🛒 Easy-to-use ordering interface
- 📱 Mobile-friendly layout
- 🎨 Attractive user interface

🛠️ Built With

- HTML5
- CSS3
- JavaScript
